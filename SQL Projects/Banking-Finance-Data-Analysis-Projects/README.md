🏦 Banking & Finance SQL Project
📌 Project Overview

This project is a Banking & Finance SQL Database Project designed to simulate real-world banking operations.
It includes customer details, branch information, transactions, and loan records to perform meaningful SQL-based analysis.

The goal of this project is to demonstrate SQL skills, database design, and analytical thinking using a banking domain dataset.

🗂️ Database Tables

The project consists of the following tables:

Customers – Stores customer personal and demographic details

Branches – Contains branch names and their respective cities

Transactions – Records deposits, withdrawals, and transfers

Loans – Tracks loan type, amount, interest rate, and loan status

Each table is structured to reflect real banking systems.

⚙️ Key SQL Concepts Used

CREATE & INSERT statements

SELECT queries

WHERE filtering

ORDER BY & GROUP BY

Aggregate functions (SUM, COUNT, AVG)

Joins between multiple tables

Conditional logic for analysis

📊 Analysis Performed

Customer distribution by city and gender

Transaction trends (deposits, withdrawals, transfers)

Loan distribution by loan type and status

Identification of active, closed, and default loans

Branch-wise customer and transaction insights

🧰 Tools & Technologies

SQL (MySQL)

Excel (for data representation & analysis)
